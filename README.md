# Gesture Sculpt

Interactive 3D object control with webcam hand gestures. Pinch to rotate; use two hands to scale. Built with Vite, React, TypeScript, Three.js, and shadcn-ui.

## Getting Started

```sh
git clone <YOUR_GIT_URL>
cd gesture-sculpt
npm install
npm run dev
```

Then open the printed local URL in your browser and allow camera access.

## Tech Stack

- Vite, React, TypeScript
- Three.js, @react-three/fiber, @react-three/drei
- Tailwind CSS, shadcn-ui

## Scripts

- dev: Start the dev server
- build: Production build
- preview: Preview the production build

## Notes

- Requires a webcam and good lighting for accurate hand tracking.
- Tested in recent Chromium-based browsers.
